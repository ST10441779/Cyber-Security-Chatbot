Cybersecurity Chatbot

📌 Description

The Cybersecurity Awareness Chatbot is a console-based application developed in C#. It educates users on safe online practices such as password security, phishing awareness, and secure browsing. The chatbot interacts with users in a friendly and engaging way, helping them understand and avoid common cybersecurity threats.



🎯 Features
	•	🔊 Voice greeting using text-to-speech (no external audio files required)
	•	🎨 ASCII art interface for enhanced visual presentation
	•	👤 Personalised interaction using the user’s name
	•	💬 Interactive chatbot with multiple cybersecurity topics:
	•	Password safety
	•	Phishing awareness
	•	Safe browsing
	•	⚠️ Input validation for empty or unknown inputs
	•	🖥️ Enhanced console UI with colours and typing effect



🛠️ Technologies Used
	•	C#
	•	.NET Framework (Console Application)
	•	System.Speech.Synthesis (Text-to-Speech)



▶️ How to Run the Project
	1.	Open the project in Visual Studio
	2.	Build the solution (Ctrl + Shift + B)
	3.	Run the application (Ctrl + F5)
	4.	Enter your name when prompted
	5.	Start interacting with the chatbot

🧠 How It Works
	•	The chatbot uses string matching to recognise user input
	•	Based on keywords, it provides relevant cybersecurity advice
	•	Input validation ensures smooth interaction without crashes
	•	Text-to-speech enhances user experience with a voice greeting
